# Sistema de notificação em NodeJs

## Descrição

Microsserviço de notificações desenvolvido no Ignite Lab de Node.js

## Rodando o app

### Desenvolvimento

#### Usando docker-compose direto

```bash
$ docker-compose up -d web
```

#### Usando Makefile

```bash
$ make run
```
